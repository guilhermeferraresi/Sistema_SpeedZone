<h1>Sistema Speed Zone</h1>

O sistema da Speed Zone foi desenvolvido usando as tecnologias do ASP.NET Core MVC., um framework moderno da Microsoft voltado para a criação de aplicações web dinâmicas, seguras e de alta performance. A arquitetura MVC (Model-View-Controller) foi adotada para organizar melhor o código, separando responsabilidades entre as camadas de modelos/classes, páginas de vizualização e controladores, facilitando a manutenção, escalabilidade e evolução do sistema.
